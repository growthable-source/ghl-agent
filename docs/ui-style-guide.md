# Voxility — Fonts & Colors (light theme)

## Fonts
- **DM Sans** — all UI text (Google Fonts)
- **DM Mono** — IDs, slugs, URLs, code (Google Fonts)

## Colors

**Brand**
- Orange `#e84425` — primary buttons, active/selected, brand

**Backgrounds**
- Page `#f8f7f4`
- Card / panel `#faf9f7`
- Inset / input / row `#f0eee9`
- Pressed / chip `#e5e2dc`

**Text**
- Primary `#1c1917`
- Secondary `#57534e`
- Tertiary (labels) `#78716c`
- Muted (placeholder/disabled) `#a8a29e`

**Borders**
- Default `#e0ddd6`
- Stronger (hover/focus) `#cdc9c0`

**Accents**
- Green `#059669` — success / positive
- Amber `#d97706` — warning / notes
- Blue `#2563eb` — info
- Red `#dc2626` — error / destructive
